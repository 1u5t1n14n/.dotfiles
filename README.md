# .dotfiles
These are the `.dotfiles` for my Arch Linux.
## OS
- OS: I use [Arch](https://archlinux.org) by the way.
- DE (kinda): [hyprland](https://github.com/hyprwm/Hyprland)
- Wallpaper Manager: [hyprpaper]()
- Display Manager: [Ly](https://github.com/fairyglade/ly)
- Terminal: [Wezterm](https://github.com/wez/wezterm)
- Launcher: [rofi](https://github.com/lbonn/rofi) (the Fork with Wayland Support)
- Browser: [Zen-Browser](https://github.com/zen-browser/desktop) (if the AUR Package isn't working for you, you need the `libpulse` Package)
- Editor: [Neovim](https://github.com/neovim/neovim)
## Other Applications
- Document Viewer: [Zathura](https://github.com/pwmt/zathura) (+ the Zathura-PDF-muPDF Package)
- System Monitor: [btop](https://github.com/aristocratos/btop)
- To-Do Manager: [on-my-mind](https://github.com/dhth/omm)
- E-Mail: [aerc](https://git.sr.ht/~rjarry/aerc)
- Music: [Spotify](https://spotify.com) (which can be used with [Spotify-Adblock](https://github.com/abba23/spotify-adblock))
- Graphics: [Inkscape](https://gitlab.com/inkscape/inkscape), [GIMP](https://gitlab.gnome.org/GNOME/gimp/) and [Krita](https://invent.kde.org/graphics/krita)

I use the [Jetbrains Mono Nerd Font](https://github.com/ryanoasis/nerd-fonts/tree/master/patched-fonts/JetBrainsMono).
